# 森罗物语：装饰
![Minecraft](https://img.shields.io/badge/Minecraft-Java_Edition-brightgreen) ![Datapack](https://img.shields.io/badge/Datapack-1.21.5~1.21.10-orange) ![License](https://img.shields.io/badge/License-GPL_v3%20|%20CC%20BY--NC--SA%204.0-blue)

森罗物语：装饰是森罗物语的一个分支模组，由数据包驱动，面向1.21.5及以上的高版本，本模组侧重森罗模型的装饰属性，舍弃复杂的功能，仅保留简单的互动。

## 模块化与构建
森罗物语：装饰按照模型主题将模组划分为了多个模块，玩家可以使用[Minecraft数据包/资源包构建工具](https://build.xekr.dev/#/KaleidoscopeMods/KaleidoscopeDeco)自主选择想要包含的模块，也可在Release中直接下载包含所有模块的完整数据包。  

模块分为主体包和扩展包，主体包主要包括装饰模型，扩展包则是在对应主体包上进行的功能拓展，玩家可按需取用。  
目前已有的模块一览：
- 主体包：
  - 玩偶(Doll)：可爱的生物玩偶，可以通过抽奖机抽取；
    - 赞助者玩偶(Credits)：包含内测赞助者的玩家玩偶；
  - 厨房(Cookery)：包含厨具、野餐用具等装饰模型；
  - 酒馆(Bar)：目前包含一个展板，**在近期会重点扩展本主题**；
  - 中式(Chinese)：包括经典的中式风格装饰，如灯笼、祈福带等；
  - 书房(Lab)：包括一些室内装饰，适合在书房或实验室等地放置的装饰；
  - 假日(Holiday)：适合度假和节日的装饰，包含小灯串、遮阳伞等装饰；
  - 街道(Road)：适合放置在室外道路的装饰，如路障、警告牌等。
- 扩展包：
  - 玩偶扩展包(Doll Plus)：为每个玩偶增加了独特的音效和粒子，并新增了一些互动，需要玩偶包作为前置；
  - 厨房扩展包(Cookery Plus)：新增了灶台和炒菜机制，新增了许多可炒制的食物；
